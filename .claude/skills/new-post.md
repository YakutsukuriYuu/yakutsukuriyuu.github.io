# 新建博客文章

当用户说要"写博客"、"写成博客"、"新建文章"、"new post"等表达要写一篇博客文章时，执行以下操作。

## 步骤

### 1. 确认信息

从用户的消息中提取或向用户确认以下信息：

- **标题**（必填）：文章标题
- **分类**（可选）：默认为 `["教程"]`，可以多选如 `["深度学习", "Python"]`
- **标签**（可选）：默认从标题和分类推断合适的标签
- **正文内容**（必填）：用户要写的博客内容

如果用户只给了标题没有给内容，就问用户想写什么内容。

### 2. 生成文件名

- 目录：`content/posts/YYYY-MM/`，其中 `YYYY-MM` 是当前年月
- 文件名：标题转 slug（全小写，空格和标点替换为 `-`，中文保持不变或拼音化）+ `.md`
- 优先保留中文标题作为文件名（和现有文章风格一致，如 `win11下的深度学习环境搭建(wsl2).md`）

### 3. 生成 frontmatter

使用 TOML 格式（`+++` 分隔），参考现有文章格式：

```toml
+++
title = '文章标题'
date = 'YYYY-MM-DDTHH:MM:SS+08:00'
draft = false
categories = ["分类1"]
tags = ["标签1", "标签2"]
author = 'Yuu'
+++
```

- `date` 使用当前时间，格式为 ISO 8601 带 +08:00 时区
- `author` 固定为 `Yuu`

### 4. 写入文件

将生成的内容写入 `content/posts/YYYY-MM/<slug>.md`，目录不存在则先创建。

### 5. 反馈

告知用户文件已创建，显示完整路径。
